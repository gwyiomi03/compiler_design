# Compiler GUI Project

## Steps to run

1. Clone the repo:
   git clone <repo_url>

2. add qt libraries
    -> name a folder .vscode and paste what's inside base the requirements.txt

3. Build and Run:
   - in your terminal vscode, paste these line by line.
   1. cd "project path/build"
   2. qmake gui.pro
   3. mingw32-make
   4. .\release\compiler_gui.exe

4. A GUI will shown
